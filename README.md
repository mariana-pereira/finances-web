# Installation

**You need to install [Node.js](https://nodejs.org/en/download/) and [Yarn](https://yarnpkg.com/) first.**

```git clone https://github.com/mariana-pereira/finances-web.git```

SSH URLs provide access to a Git repository via SSH, a secure protocol. If you have a SSH key registered in your Github account, clone the project using this command:

```git clone git@github.com:mariana-pereira/finances-web.git```

Run `yarn` to install dependencies.<br />
